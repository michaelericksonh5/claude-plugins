# Cowork Guide

Cowork plugins add skills and connectors. Verified Cowork documentation does not expose a plugin-controlled equivalent to Claude Code `settings.json` for hard model enforcement.

For Cowork users:

- Invoke `/token-saver` when starting work where token use or model choice matters.
- Start fresh chats for unrelated tasks.
- Use focused prompts that include only the files, logs, or decisions needed for the task.
- Choose cheaper models from the UI when available and appropriate.
- Use Sonnet for normal work, Haiku for simple summaries or classifications, and Opus only for difficult reasoning.

For Claude.ai and Cowork spend governance, Team and Enterprise admins should use usage analytics and extra usage limits. Treat this plugin's Cowork behavior as advisory guidance, not enforcement.
